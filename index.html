Welcome to Ping-Pong; revenge of the coding challenge!

John, and Epicodus strike back!

Input any number, and this application will return a list of all the numbers between zero and your number. But wait; there is more!! The application will locate all numbers divisible by 3, and replace them by ping. It will replace all numbers divisible by five with pong. And last but not least; all numbers divisible by fifteen will be replaced with ping-pong!
